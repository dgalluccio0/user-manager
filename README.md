# Spring Boot User Management API
Simple user manager REST API created using the Spring framework.

When the service is started, two users are created:

**ADMIN:**
* Email: `admin@galluccio.com`
* Password: `admin`

**USER:**
* Email: `user@galluccio.com`
* Password: `user`

## Live Demo
Test the API live via the interactive Swagger UI:
**[Live Swagger Documentation](https://user-manager-k711.onrender.com/usermanager/swagger-ui/index.html#/)**
